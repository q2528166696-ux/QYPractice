git init    -----git管理開始初始化
git status ----檢測當下文件狀態
git add （文件全名）管理此文件
git add .  -----管理剩餘文件
git commit -m"v1" ------提交版本
git reset --hard （版本號）----回滾
git reset --soft （版本號）-----回滾到add狀態
git reset --mix （版本號）-----回滾到紅色狀態
git reset HEAD----add之後回滾到紅
git log------查看版本
git reflog---查看所有版本
git checkout ------修改後紅回滾到綠
